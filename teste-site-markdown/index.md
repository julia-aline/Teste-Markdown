# Linha de Comando (CLI)

*CLI (Interface de Linha de Comando)* é uma forma de interagir com o computador utilizando comandos de texto, e ela está presente no dia a dia das pessoas que trabalham na aréa da tecnologia tornando o processo mais flexivél e eficiente.

## O que é CLI?

É uma interface onde o usuário digita comandos para executar ações no sistema operacional, sem usar interface gráfica.
> Ou seja, ao invés de clicar, você escreve comandos.

## Para que serve?

- Automizar tarefas
- Gerenciar arquivos e pastas
- Administrar sistemas
- Acessar sistemas remotamente

A CLI é muito usada por profissionais de tecnologia por ser rápida e eficiente.

## Como usar?

Para usar a CLI é preciso:

- Abrir o terminal
- Digitar um comando
- Pressionar Enter

Exemplo:

´´´ bash
mkdir minha_pasta
cd minha_pasta

## Mas qual seria esse *terminal*?

### Pode ser qualquer um desses:

- Prompt de Comando (CMD)
- PowerShell
- Terminal Linux
- Git Bash

# Estrutura de um comando:

Um comando normalmente possui três partes, sendo elas:

- Nome do comando
- Argumentos
- Opções (flags)
