# Linha de Comando (CLI)

*CLI (Interface de Linha de Comando)* é uma forma de interagir com o computador utilizando comandos de texto. Ela está presente no dia a dia das pessoas que trabalham na área da tecnologia tornando o processo mais flexível.

## O que é Linha de Comando?

É uma interface onde o usuário digita comandos para executar ações no sistema operacional, sem usar interface gráfica.
> Ou seja, ao invés de clicar, você escreve comandos.

## Para que serve?

- Automatizar tarefas
- Gerenciar arquivos e pastas
- Administrar sistemas
- Acessar sistemas remotamente

> A CLI é muito usada por profissionais de tecnologia por ser rápida e eficiente.

## Como usar?

Para usar a CLI é preciso:

- Abrir o terminal
- Digitar um comando
- Pressionar Enter

Exemplo:

```bash
mkdir minha_pasta
cd minha_pasta
```

## Mas o que seria esse *terminal*?

O terminal é o programa onde os comandos são digitados. Ele funciona junto com um *shell*, que é o responsável por interpretar os comandos.

### Alguns exemplos:

- Prompt de Comando (CMD)
- PowerShell
- Terminal Linux
- Git Bash

## Estrutura de um comando

Um comando normalmente possui três partes, sendo elas:

- Nome do comando
- Argumentos
- Opções (flags)

## Diferença entre CLI e GUI

|**Características**|       **CLI**       |      **GUI**      |
|-------------------|---------------------|-------------------|
| Forma de uso      | Comandos de texto   | Interface visual  |
| Facilidade        | Mais difícil        | Mais fácil        |
| Velocidade        | Mais rápida         | Mais lenta        |
| Controle          | Mais controle       | Menos controle    |
